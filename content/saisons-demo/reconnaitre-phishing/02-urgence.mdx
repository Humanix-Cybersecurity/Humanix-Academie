# Le levier d'urgence

L'urgence est l'**arme psychologique principale** du phishing. Elle court-
circuite la reflexion en activant le mode "reaction immediate". Couplee
a l'**autorite** (faux DG, DSI, banquier) et au **secret** ("ne deranger
personne, c'est confidentiel"), le combo est devastateur.

## Les signaux qui doivent declencher une alerte

| Signal | Pourquoi c'est suspect |
|---|---|
| **Delai court** ("dans l'heure", "avant 18h", "tres urgent") | Empeche la verification |
| **Heure inhabituelle** (vendredi 17h, weekend, ferie) | Bypass les controles |
| **Secret demande** ("confidentiel", "ne deranger personne") | Empeche les contre-verifications |
| **Source inhabituelle** (Gmail/Outlook perso, telephone perso, WhatsApp) | Le DG officiel n'utilise pas ces canaux |
| **Demande de virement / d'achat / de code 2FA** | Cible directe de l'attaque |
| **Ton emotionnel** ("Je compte sur toi", "Ne me decois pas") | Manipulation affective |

## La regle d'or des 30 secondes

Avant toute action sensible :
1. **Pause** : aucune urgence reelle ne justifie de sauter la verification
2. **Verification par canal alternatif** :
   - Appeler le numero pro (annuaire interne, pas celui de l'email)
   - Passer dans le bureau si c'est interne
   - Demander une confirmation Slack/Teams pro
3. **Si confirmation impossible** : escalader au superieur ou au RSSI

## Fraude au president : un cas reel

Le cas FACC Operations (2016) : 50 millions EUR transferes via phishing
de DG. Le directeur financier a virer 50 fois car le "DG" lui demandait
de faire des paiements urgents "confidentiels". Au 51e virement, il a
appele : c'etait un faux email depuis 6 mois.
