debrief: |
  Les **environnements publics** (train, cafe, aeroport,
  hotel, restaurant) sont des sources de fuites
  d'informations majeures pour les pros. Plus que le hacking,
  l'**ecoute discrete** est tres efficace et difficile a
  detecter.

  **Cas typique** : trader en train ecoute conversation DAF
  sur acquisition -> delit d'initie potentiel + violation
  confidentialite + sanctions AMF.

  **Pourquoi c'est si frequent :**

  - Pression : besoin de prendre l'appel maintenant
  - Habitude : on s'habitue a etre dans le train
  - Sentiment de "personne ne m'ecoute"
  - Manque de sensibilisation

  **Etudes (Trace International 2023)** : 47% des cadres ont
  surpris involontairement des conversations sensibles dans
  des environnements publics.

  **Les risques par environnement :**

  **Train** :
  - Voisin direct entend tout
  - Voyage de 2-3h = conversations longues
  - Wagon silencieux contourne (peu d'utilisateurs)

  **Cafe / coffee shop** :
  - Tables proches
  - Bruit ambiant masque pas la conversation a 1m
  - Visios sans casque = tout le monde entend les 2 cotes

  **Aeroport / lounge** :
  - Salons d'affaires partages avec concurrents
  - Espaces ouverts
  - Capture audio par "voisins" ciblees

  **Hotel** :
  - Lobby / restaurant
  - Couloirs
  - Salles de petit-dejeuner (proximite)

  **Restaurant** :
  - Discussions business pendant repas
  - Voisins proches
  - Serveurs entendent tout

  **Les bonnes pratiques :**

  **Pratique 1 : Differer ce qui peut l'etre**

  - Question : est-ce que cet appel peut attendre ?
  - Si oui : differer au bureau / domicile
  - Si urgence : SMS / mail crypte

  **Pratique 2 : Canaux ecrits chiffres**

  - **Signal / WhatsApp** : chiffrement bout-en-bout
  - **Mail chiffre** : pour vraies info sensibles
  - **Pas Slack** sur reseau public sans VPN

  **Pratique 3 : Voix basse + langage code**

  Si appel necessaire :
  - **Voix basse** (pas chuchote, mais discret)
  - **Pas de noms** : "le projet" au lieu de "Beta Corp"
  - **Pas de montants** : "le chiffre annonce" au lieu de
    "3.5 millions"
  - **Pas de dates** precises
  - **Casque** + micro casque (pas haut-parleur)

  **Pratique 4 : Wagon silencieux**

  - Reservation systematique pour les voyages business
  - Le wagon silencieux a moins d'utilisateurs (on respecte
    le silence)
  - **MAIS** : voix porte quand meme. Mieux : pas d'appel
    sensible meme la

  **Pratique 5 : Espaces dedies**

  - **Cabines telephoniques** dans certaines gares (Gare de
    Lyon) / lounges / WeWork
  - **Salle dediee** : conference rooms en aeroport, lounges
    business
  - **Voiture** (parking) : insonorisee, prive

  **Pratique 6 : Prevoir avant**

  - **Avant le voyage** : liste des appels prevus
  - **Identifier** ceux qui sont sensibles
  - **Differer** ou **trouver un endroit prive**

  **Pratique 7 : Bruit blanc / chiffrement audio**

  Solutions techniques (cas extremes) :
  - **Inhibiteurs de micro** : interdit en France
  - **Casques anti-bruit** : reduit l'ecoute des autres mais
    pas la portee de TA voix
  - **Apps "private mode"** : ne masquent pas la voix

  Solution la plus simple : differer.

  **Le piege "personne ne m'ecoute" :**

  Faux. Etudes :
  - **Effet cocktail party** : nous entendons toujours notre
    nom / mots-cles
  - **Voix porte** plus loin qu'on ne pense
  - **Curiosite naturelle** : les gens ecoutent les
    conversations interessantes

  Plus : parmi 50 voyageurs proches, statistiquement il y a
  des concurrents / journalistes / interesses.

  **Le piege "le casque protege" :**

  Le casque empeche TOI d'entendre les autres. Il N'EMPECHE
  PAS les autres d'entendre TA voix. Effet inverse meme :
  tu parles plus fort sans t'en rendre compte (acoustique
  du casque).

  Solution : voix basse + langage code + differer si
  vraiment sensible.

  **Le piege "delit d'initie / sanctions AMF" :**

  Pour les entreprises cotees ou avant operations financieres :
  - **Delit d'initie** : article L465-1 Code monetaire
  - **Sanctions** : 5 ans prison + 100M EUR amende
  - **AMF** investigue les coincidences suspectes

  Une fuite involontaire dans un train peut declencher une
  enquete AMF si correlation avec mouvements de marche.

  **Cas reels :**

  **Cas 2019 - DAF cabinet UK** : conversation acquisition
  ecoutee dans un train par un trader. Mouvement boursier
  suspect detecte par FCA (regulateur UK). Enquete + sanctions.

  **Cas 2022 - PME francaise** : commerciale parle au telephone
  d'un projet client (montants, planning). Voisin de cafe
  prend des notes. C'etait un consultant du concurrent.
  Negociation perdue.

  **Cas 2024 - Avocat** : conversation strategique avocat /
  client dans un avion. Stewart entend les details. Affaire
  ulterieurement vendue. Procedure deontologique.

  **La regle d'or :**

  > **Public = pas de confidentialite. Differer ce qui peut
  > l'etre, canaux ecrits chiffres pour le reste, voix basse +
  > langage code si urgence absolue. Train / cafe / aeroport /
  > hotel = environnement non-securise. Pour cadres : reflexe
  > systematique.**
