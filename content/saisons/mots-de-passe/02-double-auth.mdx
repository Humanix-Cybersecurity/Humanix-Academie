debrief: |
  La double authentification (2FA, MFA, "double facteur") est ta seconde ligne de defense. Meme si un attaquant connait ton mot de passe (fuite, phishing, devinette), il lui manque le second facteur :

  - Code envoye par SMS (faible : interceptable par SIM swap)
  - Code genere par une app (Google Authenticator, Microsoft Authenticator) → recommande
  - Cle physique (YubiKey, FIDO2) → niveau professionnel
  - Notification push sur smartphone (M365, Google) → tres bon

  En France, M365, Google, banques, reseaux sociaux : tous proposent la 2FA. Souvent en 2 clics dans les parametres. **C'est l'action cyber au plus haut ROI possible : 30 secondes pour bloquer 99 % des tentatives.**
