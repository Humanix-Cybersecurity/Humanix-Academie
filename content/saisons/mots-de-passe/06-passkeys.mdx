debrief: |
  **La passkey est la fin du mot de passe**. Cette transition, deja reelle pour
  des centaines de millions de personnes en 2026, va simplifier la vie cyber
  de tous tes collegues.

  **Comment ça marche** (en simple) :
  - Ton appareil (telephone, ordi) genere une **paire de cles** : une privee
    (qui reste sur l'appareil, jamais transmise) et une publique (envoyee au
    site).
  - Quand tu te connectes, le site envoie un **defi** que ton appareil signe
    avec la cle privee. Le site verifie la signature avec la cle publique.
  - **Aucun secret n'est jamais transmis**. Aucune base de donnees a leaker.
    Aucun phishing possible (la cle ne marche QUE sur le bon domaine).

  **Avantages concrets** :
  - **Anti-phishing native** : meme si tu cliques sur un faux Microsoft 365
    parfaitement clone, ta passkey ne s'enverra pas. Le navigateur verifie
    le domaine.
  - **Anti credential stuffing** : aucun mot de passe a reutiliser. Une
    passkey est unique par site.
  - **Plus rapide** : Touch ID / Face ID / Windows Hello = 1 seconde de
    connexion. Pas de SMS, pas de code 6 chiffres.
  - **Sync entre appareils** : iCloud Keychain (Apple), Google Password
    Manager (Android), Bitwarden (multi-plateforme) gerent la sync chiffree.

  **Sites qui supportent les passkeys en 2026** :
  - Google, Apple, Microsoft, Amazon, PayPal, GitHub, GitLab, Adobe
  - LinkedIn, X, Reddit, Discord, Notion
  - Banques : Revolut, N26, Boursorama (en cours pour la majorite)
  - Plateformes pro : 1Password, Bitwarden, Dashlane (toutes geres)
  - **Liste a jour** : https://www.passkeys.directory

  **Comment activer une passkey** : aller dans les parametres securite du
  compte, chercher "passkey" ou "cle de securite", cliquer "ajouter une
  passkey", utiliser Touch ID / Face ID / Windows Hello. C'est tout.

  La regle universelle : **active une passkey partout ou c'est possible**.
  C'est l'amelioration cyber la plus rapide a faire en 2026 - moins de 2
  minutes par site.
