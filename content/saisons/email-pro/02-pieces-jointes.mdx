debrief: |
  Les pieces jointes sont l'un des **vecteurs n°1** de malware en
  entreprise. 90% des ransomwares en France entrent par email avec une
  PJ ouverte (rapport ANSSI 2024).

  **Les 3 questions a se poser AVANT d'ouvrir une PJ (l'ordre compte) :**

  **1. J'attendais ce mail / cette PJ ?**

  Si non, suspect par defaut. Meme si l'expediteur est legitime, il
  peut etre compromis (compte hijacke). Si tu attendais une facture mais
  le format/le montant est different : suspicion.

  **2. Je connais l'expediteur ?**

  Connais = j'ai deja interagi avec, je peux verifier son existence
  (annuaire, site web, telephone). Pas connais = mefiance haute. Et
  meme connu : verifie le domaine (`@fournisseur-vincent.fr` vs
  `@fournisseur-vincent.com` typosquat).

  **3. Le format est legitime ?**

  Les formats les plus dangereux :
  - **`.exe` `.bat` `.cmd` `.scr` `.msi` `.com`** : executables
    Windows. JAMAIS legitime en facture/document. Refuse.
  - **`.pdf.exe` `.doc.exe`** : double extension. Windows masque
    `.exe` par defaut. Toujours toujours suspect.
  - **`.zip` / `.rar` / `.7z` mots de passe** : contenu cache de
    l'antivirus. Souvent malware. Refuse.
  - **Macros Office** (.docm, .xlsm, .pptm) : code executable. Demande
    confirmation a l'expediteur par appel direct AVANT d'activer.
  - **`.iso` / `.img`** : container disque. Peut contenir des
    malwares qui evitent les filtres email.
  - **`.lnk`** : raccourci Windows. Peut pointer vers une commande
    malicieuse. Refuse.

  Formats raisonnablement surs (mais pas immunises) :
  - **`.pdf`** : peut contenir des liens piegeux mais rarement du
    code executable direct.
  - **`.docx` / `.xlsx`** sans macro : ouvre dans le Reader/Lecteur
    par defaut (pas le mode "modifier").
  - **`.png` / `.jpg`** : moindre risque (pas executable).

  **Le piege Microsoft Office macros :**

  Macros = code VBA execute par Word/Excel. Vecteur classique de
  malware. Microsoft a renforce en 2022 (les macros de fichiers
  internet sont bloquees par defaut), mais l'attaquant peut demander
  a l'utilisateur de "Activer le contenu" pour faire sauter la
  protection.

  Reflex : si Word/Excel te demande "Activer les macros", **NE PAS
  ACTIVER** sans verification telephonique avec l'expediteur.

  **Si tu as ouvert une PJ suspecte :**

  1. **Deconnecte le PC du reseau** (cable Ethernet debranche, Wi-Fi
     coupe). Limite la propagation laterale.
  2. **NE redemarre PAS** (les ransomwares utilisent souvent le reboot
     pour finaliser le chiffrement).
  3. **Appelle ton DSI/IT** sur leur ligne directe (pas via le PC
     compromis qui peut etre ecoute).
  4. **Documente** ce que tu as fait (heure, fichier, action) pour
     forensic.

  **Les outils techniques de defense (cote DSI) :**

  - **Sandbox email** : Mailinblack, Vade Secure (FR), Mimecast,
    Proofpoint. Detonne les PJ suspectes dans une VM avant livraison.
  - **EDR** : HarfangLab (FR), CrowdStrike, SentinelOne. Detecte
    l'execution de malware sur le poste.
  - **Whitelist macros** : politique Group Policy qui interdit toute
    macro non signee numeriquement par l'entreprise.
  - **DMARC + SPF + DKIM** : authentification de l'expediteur (cf.
    episode 06 de cette saison).

  **L'arnaque "Facture qui sort de nulle part" :**

  Cas reel 2024 : ransomware Akira distribue par mail avec PJ
  "facture impayee" depuis un faux fournisseur. Cible : services
  comptabilite des PME francaises. Plus de 200 entreprises
  victimes en France selon CERT-FR.

  Le reflexe qui sauve : **JAMAIS de paiement urgent demande par PJ
  email sans validation hors-canal** (cf. saison Fraude au president
  pour la double validation).
