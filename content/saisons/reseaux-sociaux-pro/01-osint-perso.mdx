debrief: |
  L'**OSINT** (Open Source Intelligence) designe la collecte
  d'informations a partir de **sources publiques uniquement**.
  C'est la premiere etape de quasi toutes les attaques ciblees
  (spear phishing, fraude au president, social engineering).

  En 10 minutes, un attaquant peut etablir un profil tres
  complet de toi sans rien hacker, juste en cherchant.

  **Les sources principales d'OSINT :**

  **1. Reseaux sociaux pros**

  - **LinkedIn** : CV complet, anciennete, postes, formations,
    competences, recommandations
  - **Viadeo** : encore utilise en France
  - **Xing** : pour profils internationaux

  Souvent **publics par defaut** : meme sans etre connecte,
  acces partiel.

  **2. Reseaux sociaux personnels**

  - **Facebook** : famille, amis, voyages, opinions
  - **Instagram** : photos personnelles, geolocalisation,
    routines (lieu de cafe matinal, gym)
  - **Twitter / X** : opinions, reactions politiques, dates de
    reactions
  - **TikTok** : contexte personnel
  - **Snapchat** : moins (ephemere)

  **3. Annuaires et sources officielles**

  - **Pages Jaunes / Pages Blanches** : adresses, telephone
  - **Annuaire telephonique inverse** : numero -> nom
  - **Cadastre** : proprietes immobilieres
  - **INPI** : entreprises, dirigeants
  - **Societe.com** : comptes annuels d'entreprises
  - **Verif.com** : profil entreprises + dirigeants

  **4. Donnees fuitees (leaks)**

  - **Have I Been Pwned** : a quelles fuites tes emails
    appartiennent
  - **DeHashed** (commercial) : recherche de mots de passe
    fuites
  - **Pastebin / forums hackers** : leaks publics

  **5. Recherche image inversee**

  - **Google Images** : retrouver d'autres comptes avec la
    meme photo
  - **TinEye** : version professionnelle
  - **Yandex** : meilleur sur les visages

  **6. Sites professionnels**

  - **Site de l'entreprise** : equipe, contacts, projets
  - **Communiques de presse** : annonces, partenariats
  - **Blog d'entreprise** : auteurs, sujets

  **7. Forums / communautes**

  - **GitHub** : code, commits, projets
  - **Stack Overflow** : questions techniques (souvent revele
    le contexte de travail)
  - **Reddit** : opinions, hobbies
  - **Forums specialises** : centres d'interet

  **8. Geolocalisation**

  - **Metadata de photos** : EXIF GPS si non-supprime
  - **Check-in Facebook / Instagram**
  - **Strava** : trajets de course / velo (incident militaire
    US 2018)
  - **Google Maps reviews** : restaurants frequentes

  **9. Sources techniques (pour devs)**

  - **WHOIS** : qui a achete tel domaine
  - **GitHub** : code public + commits avec emails
  - **Shodan** : devices connectes a l'IP de l'entreprise

  **10. Wayback Machine**

  - Versions anciennes de sites web
  - Permet de voir des informations supprimees recemment

  **Outils OSINT (legaux et utilises par redteamers) :**

  - **Maltego** : visualisation des liens entre personnes /
    entreprises
  - **theHarvester** : collecte massive (emails, sous-domaines)
  - **SpiderFoot** : automatisation OSINT
  - **OSINT Framework** : annuaire de tous les outils
  - **Sherlock** : recherche d'un username sur 300+ sites
  - **GHunt** : info sur compte Google a partir d'email

  **Comment l'attaquant utilise l'OSINT :**

  **Usage 1 : Spear phishing**

  L'attaquant connait ton parcours, ton manager, tes projets
  recents. Il envoie un mail :

  > "Salut [ton prenom], suite a notre echange chez Grand
  > Cafe la semaine derniere [reel : il a vu ton check-in
  > Instagram], je t'envoie le compte-rendu du projet [reel :
  > nom de projet sur LinkedIn]. Clique ici pour la version
  > finale."

  Credibilite +++ -> taux de clic >50% selon studies.

  **Usage 2 : Reset de mot de passe**

  Beaucoup de sites demandent encore des "questions secretes" :
  - Nom de jeune fille de la mere : facile (Facebook ta mere)
  - Premiere ecole : LinkedIn / Facebook
  - Nom de ton premier animal : Instagram

  L'attaquant peut prendre le controle de comptes via reset.

  **Usage 3 : Fraude au president**

  L'attaquant connait :
  - Le nom du DG / DAF
  - Sa situation actuelle (en deplacement -> via LinkedIn ou
    Twitter)
  - Le contexte business

  Il appelle le comptable en se faisant passer pour le DG :

  > "Je suis a Tokyo pour un deal urgent confidentiel, fais ce
  > virement de 80K EUR sur ce compte..."

  Cf saison fraude-president.

  **Usage 4 : Ingenierie sociale physique**

  L'attaquant connait :
  - Les locaux (Google Maps, photos LinkedIn)
  - Les noms des employes (LinkedIn de l'entreprise)
  - Les fournisseurs (page Web, communiques)

  Il se presente comme un faux technicien (cf saison
  acces-physiques ep 03) avec un context credible.

  **Usage 5 : Vol d'identite**

  Toutes les infos OSINT permettent :
  - Ouverture de comptes en banque (avec vol de papiers
    complementaires)
  - Souscription credits
  - Demarches administratives au nom de la victime

  **Reduire son empreinte OSINT (audit perso) :**

  **Etape 1 : Auto-recherche**

  Cherche-toi :
  - Google : "[Ton nom] [ta ville]"
  - Bing : meme requete
  - Sherlock (online) : ton username habituel
  - Have I Been Pwned : ton email

  Note ce qui sort.

  **Etape 2 : LinkedIn**

  Reglages a verifier :
  - **Profile public** : limite a "votre nom et votre photo"
  - **Visibility of activity** : connexions seulement
  - **Recherche par moteurs** : desactiver
  - **Email associe** : pas perso (si possible)
  - **Photo profil** : OK mais pas trop revelateur

  **Etape 3 : Facebook**

  - **Audience par defaut** : Amis (pas Public)
  - **Recherche par email/telephone** : friends only
  - **Photos taggees** : approbation manuelle
  - **Audit historique** : Facebook outil pour rendre prive
    les anciens posts
  - **Geolocalisation** : desactiver pour photos

  **Etape 4 : Instagram**

  - **Compte prive** : recommande
  - **Geolocalisation** : desactiver
  - **Sharing automatique** vers Facebook : desactiver
  - **Stories** : seulement amis proches pour les sensibles

  **Etape 5 : Twitter / X**

  - **Compte prive** ou cleanup des vieux tweets (TweetDelete,
    TweetEraser)
  - **Geolocalisation** : desactivee
  - **Email caches**

  **Etape 6 : Donnees fuitees**

  - **Have I Been Pwned** : changer mots de passe pour comptes
    fuites
  - **MFA** sur tous les comptes critiques
  - **Mots de passe uniques** (gestionnaire)

  **Etape 7 : Annuaires**

  - **Pages Jaunes / Pages Blanches** : demander suppression
  - **Verif.com / Societe.com** : difficile a faire enlever
    (donnees publiques d'entreprises)

  **Etape 8 : Photos / metadata**

  - **Avant publication** : supprimer EXIF (Photoshop, GIMP,
    Mac Preview)
  - **Apres publication** : pas grand chose a faire (les
    plate-formes les retirent souvent)

  **Etape 9 : Alertes**

  - **Google Alerts** : alerte "[Ton nom]" par email
  - **Detection nouvelle apparition** rapide
  - **Reaction** rapide possible

  **Etape 10 : Education famille**

  Ta famille / amis peuvent te tagguer / poster avec toi.
  Sensibiliser :
  - Pas de geolocalisation
  - Pas de check-in en temps reel
  - Demander avant de tagger / poster

  **Pour les profils sensibles (cadres, dirigeants, RSSI) :**

  Mesures renforcees :
  - **Profil LinkedIn minimal** : nom + entreprise actuelle
    seulement
  - **Pas de Facebook public**
  - **Pas de Twitter perso** (creer un alias)
  - **VPN** systematique
  - **Email pro** pour actes engageants seulement
  - **Telephone separe** perso / pro

  **Le piege "j'ai rien a cacher" :**

  Argument classique mais errone. L'OSINT n'est pas une
  question de "cacher", mais de **reduire la surface
  d'attaque** :

  - Plus on sait sur toi = plus le phishing est credible
  - Plus on sait sur toi = plus le reset de mot de passe est
    facile
  - Plus on sait sur toi = plus la fraude au president est
    credible

  Memes les "informations innocentes" peuvent etre exploitees.

  **Le piege "Facebook c'est entre amis" :**

  Tu as 200 amis Facebook. Combien d'eux :
  - Ont un compte compromis (mot de passe reutilise) ?
  - Sont des "faux amis" (faux profils) ?
  - Partagent les contenus avec leurs amis (cascade) ?

  La reponse statistique : 5-10% des comptes Facebook sont
  compromis ou faux a un moment donne. Ton "espace prive"
  fuit constamment.

  **Le piege "je suis trop petit pour etre une cible" :**

  L'OSINT est **automatise**. Les attaquants scrapent
  massivement et choisissent leurs cibles parmi les profils
  exploitables. Tu peux devenir cible sans aucune raison
  particuliere.

  Plus : si tu as acces a des donnees sensibles dans ton
  travail (paie, IBAN, donnees clients), tu **es** une cible.

  **Cas reels :**

  **Cas 1 - Mati Aharoni 2014 (Strava military leak)**

  Strava (app de course/velo) publiait par defaut les trajets.
  Bilan : tracks de soldats US dans des bases secretes au
  Moyen-Orient identifiables. Bases compromises par OSINT
  publique.

  **Cas 2 - PME Lyon 2023 (FOVI 80K EUR)**

  L'attaquant a fait OSINT sur le DG (LinkedIn -> deplacement
  Tokyo en cours). Faux email "DG depuis Tokyo, urgent
  paiement". Le comptable, n'ayant aucun moyen de douter
  (l'OSINT etait bien fait), a paye.

  **Cas 3 - Personne celebre 2024 (vol d'identite via OSINT)**

  Une personne moyennement connue a vu son identite usurpee
  pour ouvrir 5 comptes bancaires. L'attaquant avait
  reconstruit toute son identite via OSINT (date naissance,
  nom mere, etc.).

  **La regle d'or :**

  > **L'OSINT est l'etape 1 de la plupart des attaques
  > ciblees. Reduction de l'empreinte = reduction du risque.
  > Audit perso 1x/an + reglages confidentialite + alertes
  > Google + education famille. L'objectif est PROPORTIONNE,
  > pas de devenir invisible.**
