debrief: |
  Le **faux recruteur sur LinkedIn** est l'une des techniques
  d'ingénierie sociale les plus repandues. Cible :

  - **Vol d'identité** (RIB, carte d'identité, CV)
  - **Vol de propriété intellectuelle** (CV detaille, projets)
  - **Cartographie d'entreprise** (qui fait quoi, hierarchie)
  - **Phishing avance** (faux entretien, faux site)
  - **Recrutement de "taupe"** (employé convaincu de
    travailler pour le concurrent)

  **LinkedIn 2024** : ~10 millions de faux profils detectes
  par mois. Beaucoup ont l'apparence de profils legitimes.

  **Les 8 signes d'un faux recruteur :**

  **1. Profil recent ou peu actif**

  - **Date de création** : moins de 6 mois (alerte)
  - **Posts** : peu nombreux, generiques, copies/colles
  - **Activite** : likes / comments rares
  - **Connexions** : croissance soudaine et massive

  **Vérification** : timeline d'activite + premier post.

  **2. Photo de profil suspecte**

  - **Photo trop pro** (mannequin, photo banque images)
  - **Recherche image inversee** (Google Images / Yandex) :
    si elle apparait dans plusieurs profils, fake
  - **Photos generees par IA** : détection difficile mais
    parfois imperfections (asymetrie yeux, oreilles)

  **3. Description generique**

  - "Senior recruiter passionne par les talents"
  - "Connecteur de talents et d'opportunites"
  - **Pas de specialite** (industrie, type de poste)
  - **Pas de localisation precise**

  **Vraie recruteuse** : decrit son secteur, son entreprise,
  sa methodologie.

  **4. Entreprise qui n'existe pas / generique**

  - **TechCorp**, **Global Solutions**, **Strategy Partners** :
    noms generiques alarmants
  - **Vérification** : site web legitime ? Numéro SIRET en
    France ? LinkedIn entreprise serieux ?

  **5. Salaire / conditions trop attractifs**

  - **+50% de ton salaire actuel** sans explication
  - **Remote total + flex hours + tickets restaurants** -
    "tout, partout"
  - **Postes senior** sans entretien preliminaire

  Si trop beau pour être vrai = c'est faux.

  **6. Demande de documents sensibles avant entretien**

  - **RIB** : aucune raison legitime AVANT signature contrat
  - **Carte d'identité / passeport** : ne se demande qu'a la
    signature, pas avant
  - **Numéro sécurité sociale** : jamais avant l'embauche
  - **Mot de passe** : jamais (vraie alerte si demande)
  - **Acces aux systèmes actuels** : aucun recruteur n'en a
    besoin

  **7. Pression / urgence**

  - **"Décision sous 24h"**
  - **"Plusieurs candidats"** alors que c'était un message non
    sollicite
  - **"Je ne peux pas attendre"**

  Vrais recruteurs : pas presses, methodologie claire.

  **8. Migration vers d'autres canaux**

  - **WhatsApp / Telegram** demande rapidement
  - **Email perso** au lieu d'email d'entreprise
  - **Sortir de LinkedIn** dès le premier message

  Vrais recruteurs : restent sur LinkedIn / email pro
  d'entreprise.

  **Comment vérifier un recruteur :**

  **Étape 1 : Profil LinkedIn**

  - Date de création (avant message d'introduction)
  - Activite reguliere (posts, comments, likes sur >1 an)
  - Cohérence expérience / formations
  - Recommandations reçues / données

  **Étape 2 : Entreprise**

  - Site web (vérifier authenticite, age du domaine via WHOIS)
  - Numéro SIREN / RCS (société.com pour France)
  - LinkedIn entreprise (anciennete, employés actuels)
  - Glassdoor / Indeed reviews
  - Google search "entreprise X scam" / "entreprise X arnaque"

  **Étape 3 : Profil croise**

  - Connexions communes (qui peut me parler de cette personne)
  - Recommandations validables (anciens collègues)
  - Cohérence parcours / poste actuel

  **Étape 4 : Demande d'entretien téléphonique**

  - "Avant tout document, pourrions-nous avoir un entretien
    téléphonique ?"
  - **Vrai recruteur** : accepte volontiers
  - **Faux recruteur** : esquive, redirige vers email / WhatsApp

  **Étape 5 : Vérification recursive**

  Si tout passe :
  - Demande nom de votre futur manager
  - Demande nom du DRH de l'entreprise
  - Verifie sur LinkedIn que ces personnes existent
  - Si possible : appel a la standard de l'entreprise pour
    confirmer

  **Que peut faire un faux recruteur :**

  **a) Vol d'identité directe**

  - RIB + carte d'identité + adresse = ouverture comptes
    bancaires
  - Souscription credits
  - Demarches administratives au nom de la victime

  **Cout typique** : 5-50K EUR + 1-3 ans de procédures.

  **b) Phishing avance / faux entretien**

  - "Telechargez notre app pour l'entretien video"
  - L'app contient un malware
  - Acces complet a ton téléphone / PC

  **c) Recrutement de "taupe"**

  - "Vous etes super, on aimerait que vous restiez chez votre
    employeur actuel mais nous donnez quelques infos
    intérieures"
  - "Bonus mensuel pour info confidentielle"
  - C'est de l'**espionnage industriel**, illegal.

  **d) Vol de propriété intellectuelle**

  - "Envoyez-nous un exemple de votre travail"
  - "Vos meilleures realisations en détail"
  - Le faux recruteur revend ces infos a un concurrent.

  **e) Cartographie d'entreprise**

  - "Qui est votre manager ?"
  - "Qui s'occupe du projet X chez vous ?"
  - "Combien de personnes sur le sujet Y ?"
  - Servira pour spear phishing futur.

  **f) Réseau de relations**

  - "Pourriez-vous me presenter quelques collègues que vous
    estimez ?"
  - Le faux recruteur agrandit son réseau de cibles.

  **Cas spécifiques :**

  **Cas 1 : Faux recruteur agence**

  L'attaquant se fait passer pour un cabinet de chasseur de
  tête (ex. "Spencer Stuart", "Heidrick Struggles" - vrais
  cabinets connus). Mais :
  - Email perso (pas spencerstuart.com)
  - Numéro perso
  - Demandes inhabituelles

  Solution : appel a la vraie agence pour vérifier l'identité
  du recruteur.

  **Cas 2 : Faux recruteur interne**

  L'attaquant se fait passer pour le RH **interne** de ton
  entreprise actuelle pour ajouter de la crédibilité. Email
  type "alice.dupont@ton-entreprise-fr.com" (typosquatting)
  ou compte LinkedIn genere.

  Solution : vérifier en intern (Slack, Teams, téléphone
  bureau) avant de répondre.

  **Cas 3 : Faux recruteur startup tech**

  Profil avec photo "lifestyle business", "I scout the best
  talents for the next unicorn", recommendation par d'autres
  faux profils.

  Solution : vérification société + entretien téléphonique
  en premier.

  **Cas 4 : Faux recruteur "intelligence agency"**

  L'attaquant se fait passer pour un recruteur d'une agence
  de renseignement (DGSE, MI6, CIA). Demande très spécifique :
  ton expérience cyber + ton acces actuel.

  Solution : ces agences ne recrutent JAMAIS via LinkedIn
  cold message. Si interest reel : approche via parcours
  formel (concours, candidature spontanee).

  **Procédure de signalement si suspect :**

  1. **Capture d'écran** du message + profil
  2. **Signaler** sur LinkedIn (bouton "Signaler")
  3. **Bloquer** le profil
  4. **Si vol identité** : plainte penale + Cybermalveillance.
    gouv.fr
  5. **Informer ton manager / RSSI** : peut faire partie d'une
    campagne ciblant ton entreprise

  **Configuration LinkedIn defensive :**

  Pour reduire l'exposition :

  - **Contact info** : email pro non visible
  - **Phone** : non visible
  - **Activity broadcast** : selectif
  - **Profile visibility outside LinkedIn** : limite
  - **Connections only** : qui peut t'envoyer message
  - **Demand confirmation** pour ajouter aux connexions

  **Le piege "j'aurais reconnu un faux profil" :**

  Etudes : même les pros cyber se font avoir 10-15% du temps
  (tests internes). Les attaquants soignent leurs profils.
  Ne pas se baser sur l'intuition = methodologie rigoureuse.

  **Le piege "ce sera plus rapide de fournir tout vite" :**

  L'urgence est exactement le levier de manipulation. Plus le
  recruteur insiste pour avoir tes documents rapidement, plus
  c'est suspect.

  **Cas reel : LinkedIn 2024 (campagne nord-coreenne)**

  Opération **DREAM JOB** (rapport US Treasury 2024) :
  - Faux recruteurs nord-coreens ciblant des employés IT
  - Salaires attractifs proposes
  - Faux entretiens video avec deepfake
  - Résultat : embauche d'agents nord-coreens dans entreprises
    US (cf saison cyber-rh ep 04)

  Et : **DREAM JOB cible** ciblant des employés existants pour
  les recruter comme taupes (vol code source, exfiltration
  données).

  **Cas reel : PME française 2023 (vol d'identité)**

  Cadre IT a reçu message d'un "recruteur chez Capgemini".
  Profil convaincant, salaire +30%. A envoye RIB + CI pour
  "preparer son contrat". Ouverture frauduleuse de 3 comptes
  bancaires + souscription credits.

  Cout : 25K EUR + 2 ans de procédures.

  **La règle d'or :**

  > **Aucun recruteur legitime ne demande RIB / CI / mot de
  > passe AVANT signature contrat. Vérification systematique :
  > profil ancien + entreprise reelle + entretien
  > téléphonique. Pression / urgence = signal d'alerte. En cas
  > de doute : ignorer ou signaler.**
