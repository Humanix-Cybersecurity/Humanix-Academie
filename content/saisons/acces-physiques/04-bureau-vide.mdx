debrief: |
  Le **bureau vide non securise** est l'une des fuites les plus
  silencieuses et fréquentes en entreprise. Pas de hacker, pas de
  malware - juste une absence de discipline élémentaire qui ouvre la
  porte a l'observation, la photographie, le vol.

  **La 'clean desk policy' (bureau range) :**

  Norme ISO 27001 (contrôle A.7.7) impose une politique de bureau
  range pour les organisations certifiees. Principe : quand tu
  quittes ton poste (même 5 minutes), aucune information sensible ne
  doit être visible.

  **Les 4 catégories d'information a securiser :**

  **1. Écran (le plus immediat)**

  ❌ Écran ouvert avec mails / docs / chat = lisible par tout passant.
  ✅ Écran verrouille (Windows : Win+L. macOS : Cmd+Ctrl+Q. Linux :
     Super+L).

  Le verrouillage prend < 1 seconde. Pas d'excuse.

  **2. Documents papier**

  ❌ Pile de "Résultats commerciaux 2026 confidentiels" sur le
     bureau = lisible.
  ✅ Documents ranges dans un tiroir ferme a cle, ou dans un
     classeur avec couverture neutre.

  Le menage / les visiteurs / les collègues curieux peuvent lire
  rapidement (5-10 secondes par feuille en photo).

  **3. Post-it / notes manuscrites**

  ❌ Post-it sur l'écran "mdp Outlook : Lille75!@" = donnez les
     cles.
  ✅ Mot de passe dans un gestionnaire de mots de passe
     (Bitwarden, 1Password, KeePass), JAMAIS ecrit physiquement.

  Si tu DOIS noter un mot de passe (rare cas) : dans un coffre-fort
  ferme a cle, jamais visible.

  **4. Matériel**

  ❌ Cle USB sur le bureau, smartphone deverrouille pose, badge
     visible.
  ✅ Tout dans un sac ou tiroir ferme a cle pendant les pauses.

  **Verrouillage automatique (le filet de sécurité) :**

  Configure ton ordinateur pour verrouiller automatiquement après
  X minutes d'inactivite :

  - **Windows** : Settings > Personalization > Lock Screen > Screen
    timeout : 5 minutes max.
  - **macOS** : Reglages > Écran de verrouillage > "Demander le mot
    de passe immédiatement" + délai 5 minutes.
  - **Linux** : depend du DE (GNOME, KDE), généralement Settings >
    Privacy > Screen Lock.

  En entreprise, le MDM (cf. saison mobile-smartphone) peut imposer
  ce verrouillage automatique a tous les postes.

  **La verrouillage manuel (l'habitude a créer) :**

  Apprends-toi le raccourci :
  - **Windows** : Win + L
  - **macOS** : Cmd + Ctrl + Q
  - **Chrome OS** : Search + L

  En une seconde, l'écran est verrouille. C'est plus rapide que de
  prendre ton cafe.

  **Le piege "je connais mes collègues, ils ne regardent pas" :**

  FAUX dans 3 cas :
  - **Visiteurs** : tu ne contrôles pas qui passe dans ton open-space.
  - **Menage / techniciens** : peuvent voler / photographier sans
    laisser de trace.
  - **Collegues curieux ou malveillants** : la majorité est honnête,
    mais 1 sur 100 a des problèmes financiers / chantage / curiosite
    excessive.

  Pour les RH / DAF / DPO : tes documents sont legalement sensibles
  (données personnelles employés, informations financieres). La
  fuite par bureau ouvert peut être une violation RGPD.

  **Cas reels :**

  - **2018 - Banque française** : un employé a laisse son écran
    ouvert avec une feuille Excel "Liste des dossiers en cours"
    visible. Un visiteur (faux client) a photographie l'écran.
    Plainte client + amende interne.
  - **2020 - Cabinet d'avocats** : un avocat avait un post-it avec
    mot de passe collecte sur son écran. Le menage a vu, l'a partage
    avec un proche. 3 dossiers clients ont fuite. Cabinet poursuivi.
  - **2022 - Hôpital français** : photo d'écran d'un PC infirmier
    laisse ouvert (avec données patients) postee sur Twitter par un
    employé de menage (qui voulait illustrer son métier). Procédure
    CNIL en cours.

  **Le piege "je travaille tard, plus personne n'est la" :**

  - 19h-7h : menage, sécurité, peut-être des collègues qui
    travaillent tard.
  - Weekend : agents d'entretien, mainteneurs.
  - Locaux ouverts 24/7 (centres de service) : encore plus
    d'exposition.

  Le bureau ferme a cle est rarement vraiment ferme. Considere
  qu'il y a toujours quelqu'un qui peut passer.

  **Cas particulier - les imprimantes :**

  ❌ Imprimer un document confidentiel et oublier de le recuperer
     immédiatement.
  ✅ Imprimer en mode "secure print" (PIN code) ou recuperer
     immédiatement.

  Les imprimantes sont aussi des points de fuite : papiers laisses
  empiles dans le bac peuvent être vus / photographies par
  n'importe qui.

  **Pour un bureau partage / open-space :**

  Sensibilisation visuelle :
  - Affiche "Avant de partir : verrouille ton écran (Win+L)" pres
    des espaces communs
  - Exercice "test de la photo d'écran" : un volontaire (RSSI ou
    consultant) prend en photo les écrans non verrouilles pendant
    une journée = sensibilisation par l'expérience

  **La règle d'or :**

  > **Un bureau quitte = un bureau verrouille et range.** Même 5
  > minutes. Même pour un cafe.
  >
  > Le verrouillage prend 1 seconde. Le rangement, 30 secondes. Le
  > risque sans, c'est des données personnelles, des secrets
  > commerciaux, des credentials. Le ROI est évident.

  **Configurer maintenant :**

  - Verrouillage auto 5 min sur tous tes appareils
  - Apprendre le raccourci Win+L / Cmd+Ctrl+Q
  - Gestionnaire de mots de passe (jeter tous tes post-it)
  - Tiroir ferme a cle pour les documents sensibles
