debrief: |
  Le **bureau vide non securise** est l'une des fuites les plus
  silencieuses et frequentes en entreprise. Pas de hacker, pas de
  malware - juste une absence de discipline elementaire qui ouvre la
  porte a l'observation, la photographie, le vol.

  **La 'clean desk policy' (bureau range) :**

  Norme ISO 27001 (controle A.7.7) impose une politique de bureau
  range pour les organisations certifiees. Principe : quand tu
  quittes ton poste (meme 5 minutes), aucune information sensible ne
  doit etre visible.

  **Les 4 categories d'information a securiser :**

  **1. Ecran (le plus immediat)**

  ❌ Ecran ouvert avec mails / docs / chat = lisible par tout passant.
  ✅ Ecran verrouille (Windows : Win+L. macOS : Cmd+Ctrl+Q. Linux :
     Super+L).

  Le verrouillage prend < 1 seconde. Pas d'excuse.

  **2. Documents papier**

  ❌ Pile de "Resultats commerciaux 2026 confidentiels" sur le
     bureau = lisible.
  ✅ Documents ranges dans un tiroir ferme a cle, ou dans un
     classeur avec couverture neutre.

  Le menage / les visiteurs / les collegues curieux peuvent lire
  rapidement (5-10 secondes par feuille en photo).

  **3. Post-it / notes manuscrites**

  ❌ Post-it sur l'ecran "mdp Outlook : Lille75!@" = donnez les
     cles.
  ✅ Mot de passe dans un gestionnaire de mots de passe
     (Bitwarden, 1Password, KeePass), JAMAIS ecrit physiquement.

  Si tu DOIS noter un mot de passe (rare cas) : dans un coffre-fort
  ferme a cle, jamais visible.

  **4. Materiel**

  ❌ Cle USB sur le bureau, smartphone deverrouille pose, badge
     visible.
  ✅ Tout dans un sac ou tiroir ferme a cle pendant les pauses.

  **Verrouillage automatique (le filet de securite) :**

  Configure ton ordinateur pour verrouiller automatiquement apres
  X minutes d'inactivite :

  - **Windows** : Settings > Personalization > Lock Screen > Screen
    timeout : 5 minutes max.
  - **macOS** : Reglages > Ecran de verrouillage > "Demander le mot
    de passe immediatement" + delai 5 minutes.
  - **Linux** : depend du DE (GNOME, KDE), generalement Settings >
    Privacy > Screen Lock.

  En entreprise, le MDM (cf. saison mobile-smartphone) peut imposer
  ce verrouillage automatique a tous les postes.

  **La verrouillage manuel (l'habitude a creer) :**

  Apprends-toi le raccourci :
  - **Windows** : Win + L
  - **macOS** : Cmd + Ctrl + Q
  - **Chrome OS** : Search + L

  En une seconde, l'ecran est verrouille. C'est plus rapide que de
  prendre ton cafe.

  **Le piege "je connais mes collegues, ils ne regardent pas" :**

  FAUX dans 3 cas :
  - **Visiteurs** : tu ne controles pas qui passe dans ton open-space.
  - **Menage / techniciens** : peuvent voler / photographier sans
    laisser de trace.
  - **Collegues curieux ou malveillants** : la majorite est honnete,
    mais 1 sur 100 a des problemes financiers / chantage / curiosite
    excessive.

  Pour les RH / DAF / DPO : tes documents sont legalement sensibles
  (donnees personnelles employes, informations financieres). La
  fuite par bureau ouvert peut etre une violation RGPD.

  **Cas reels :**

  - **2018 - Banque francaise** : un employe a laisse son ecran
    ouvert avec une feuille Excel "Liste des dossiers en cours"
    visible. Un visiteur (faux client) a photographie l'ecran.
    Plainte client + amende interne.
  - **2020 - Cabinet d'avocats** : un avocat avait un post-it avec
    mot de passe collecte sur son ecran. Le menage a vu, l'a partage
    avec un proche. 3 dossiers clients ont fuite. Cabinet poursuivi.
  - **2022 - Hopital francais** : photo d'ecran d'un PC infirmier
    laisse ouvert (avec donnees patients) postee sur Twitter par un
    employe de menage (qui voulait illustrer son metier). Procedure
    CNIL en cours.

  **Le piege "je travaille tard, plus personne n'est la" :**

  - 19h-7h : menage, securite, peut-etre des collegues qui
    travaillent tard.
  - Weekend : agents d'entretien, mainteneurs.
  - Locaux ouverts 24/7 (centres de service) : encore plus
    d'exposition.

  Le bureau ferme a cle est rarement vraiment ferme. Considere
  qu'il y a toujours quelqu'un qui peut passer.

  **Cas particulier - les imprimantes :**

  ❌ Imprimer un document confidentiel et oublier de le recuperer
     immediatement.
  ✅ Imprimer en mode "secure print" (PIN code) ou recuperer
     immediatement.

  Les imprimantes sont aussi des points de fuite : papiers laisses
  empiles dans le bac peuvent etre vus / photographies par
  n'importe qui.

  **Pour un bureau partage / open-space :**

  Sensibilisation visuelle :
  - Affiche "Avant de partir : verrouille ton ecran (Win+L)" pres
    des espaces communs
  - Exercice "test de la photo d'ecran" : un volontaire (RSSI ou
    consultant) prend en photo les ecrans non verrouilles pendant
    une journee = sensibilisation par l'experience

  **La regle d'or :**

  > **Un bureau quitte = un bureau verrouille et range.** Meme 5
  > minutes. Meme pour un cafe.
  >
  > Le verrouillage prend 1 seconde. Le rangement, 30 secondes. Le
  > risque sans, c'est des donnees personnelles, des secrets
  > commerciaux, des credentials. Le ROI est evident.

  **Configurer maintenant :**

  - Verrouillage auto 5 min sur tous tes appareils
  - Apprendre le raccourci Win+L / Cmd+Ctrl+Q
  - Gestionnaire de mots de passe (jeter tous tes post-it)
  - Tiroir ferme a cle pour les documents sensibles
