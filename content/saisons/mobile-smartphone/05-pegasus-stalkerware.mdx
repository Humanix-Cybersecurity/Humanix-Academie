debrief: |
  Pegasus et le stalkerware sont 2 categories tres differentes de
  malwares mobiles, souvent confondus mais distincts.

  **Pegasus (NSO Group, Israel) - surveillance commerciale haut de
  gamme :**

  - Vendu (cher, ~5M USD pour 50 cibles) a des etats clients de NSO
    (souvent autoritaires)
  - **0-click exploit** : pas besoin que la cible clique sur quoi que
    ce soit. Un message iMessage suffit pour infecter.
  - Capacites totales : lecture mails/messages, audio, video,
    localisation, exfiltration apps (WhatsApp, Signal compris).
  - Zero day exploits a disposition (Apple/Google patchent en
    permanence, NSO trouve de nouveaux 0-day)
  - Cibles confirmees : journalistes (Khashoggi avant assassinat,
    plusieurs collaborateurs Le Monde, Mediapart), avocats des
    droits humains, activistes, dissidents, dirigeants (chefs
    d'etat europeens cibles dont Macron en 2021).

  **Stalkerware - surveillance domestique installee physiquement :**

  - Categorie d'apps "espion" vendues a tout consommateur ($30-300/an)
  - Marques : mSpy, Cocospy, FlexiSpy, Spyzie, Highster Mobile,
    Hoverwatch
  - Necessite **acces physique** au tel cible pour installation
    (5-15 min)
  - Capacites variables : lecture SMS, calls, GPS, parfois apps
    chiffrees
  - Utilises principalement par : **conjoint suspicieux**, parent
    "control parental" abusif, employeur abusif
  - Souvent installe pendant que la cible dort/douche
  - Tres present dans les contextes de violence conjugale : 30% des
    victimes de violence ont du stalkerware sur leur tel selon
    Avast 2023.

  **Comment differencier :**

  | Critere | Pegasus | Stalkerware |
  |---|---|---|
  | Vecteur d'infection | 0-click (iMessage, etc.) | Acces physique |
  | Cible | Journalistes, activistes, decideurs | Tout le monde, conjoint/famille |
  | Cout | 5M USD pour state actor | $30-300/an |
  | Detection difficulte | Tres dur (camoufle) | Moyen (souvent une icone reste) |
  | Persistence reboot | Variable | Oui (app installee) |
  | Cible typique | < 1000 personnes par etat | Millions |

  **Les signes (a surveiller) :**

  **Signes COMMUNS (Pegasus + stalkerware) :**
  - **Batterie qui tient moins** : malware tourne en fond
  - **Tel chaud** : process en arriere-plan
  - **Consommation data anormale** : exfiltration constante
  - **Apps inconnues** : dans la liste des apps installees
  - **Ralentissement** general

  **Signes specifiques Pegasus :**
  - **Apps qui crashent** sans raison (variantes Pegasus
    instables) - notamment le navigateur, FaceTime
  - **Notifications push de iMessage / WhatsApp** que tu ne
    reconnais pas
  - **Appareil qui semble communiquer** meme quand l'ecran est
    eteint

  **Signes specifiques stalkerware :**
  - **Icone d'app etrange** dans le menu (souvent "System
    Service", "WiFi Helper", "Battery Manager", icone discrete)
  - **Permissions admin device** activees pour une app que tu
    ne reconnais pas (Settings > Security > Device Admin)
  - **Compte Google/Apple** non identifie connecte

  **Comment detecter (les outils) :**

  **Pour Pegasus - Mobile Verification Toolkit (MVT) :**

  - Outil officiel **gratuit** developpe par Amnesty International
    et Citizen Lab.
  - Disponible sur GitHub : github.com/mvt-project/mvt
  - Marche pour **iOS** (a partir d'une sauvegarde iTunes/Mac) et
    **Android** (a partir du tel directement).
  - Detecte les Indicators of Compromise (IOC) connus de Pegasus
    (domaines, signatures, comportements).
  - Necessite des compétences techniques (CLI Python). Pour les
    non-techniques : Amnesty propose une assistance gratuite.

  **Pour stalkerware :**

  - **iOS** : tres difficile a installer (jailbreak necessaire pour
    la plupart des stalkerware). Si ton iPhone est jailbreak sans
    que tu le sache = signe rouge.
  - **Android** : installation plus facile via APK sideloading.
    - **MVT** detecte aussi certains stalkerware
    - **Sticker (TinyCheck)** outil gratuit du CERT-FR pour
      detecter le trafic de stalkerware
    - **Antivirus** (Bitdefender, Kaspersky, Sophos) detectent les
      stalkerware connus
    - Verification manuelle : Settings > Apps > liste de toutes les
      apps installees + permissions

  **Comment se proteger :**

  **Pour les profils a haut risque (journalistes, dissidents,
  activistes) :**

  1. **Mises a jour OS immediates** : iOS 18, Android 14+. Les
     patches Apple/Google bouchent les 0-day Pegasus connus.
  2. **iOS Lockdown Mode** : Reglages > Confidentialite et Securite
     > Mode Lockdown. Reduit drastiquement la surface d'attaque
     (desactive iMessage exotique, FaceTime non-contacts, etc.).
     Apple a cree ce mode specifiquement pour les cibles Pegasus.
  3. **Tel dedie** : un tel pro pour les contacts sensibles, un tel
     perso pour le reste. Limiter les communications sensibles a
     un seul appareil.
  4. **Signal** au lieu de iMessage/WhatsApp pour les communications
     sensibles.
  5. **Reboot quotidien** : redemarrer le tel chaque nuit. Certaines
     variantes Pegasus ne sont pas persistantes au reboot.
  6. **Contact Amnesty Tech** ou **Reporters Without Borders** pour
     audit confidentiel gratuit.

  **Pour proteger contre le stalkerware (cas violence conjugale) :**

  1. **Audit des apps installees** : si une app que tu ne reconnais
     pas est presente, suspect.
  2. **Verifier les Device Admin** sur Android : Settings > Security
     > Device Admin. Aucune app inconnue ne devrait etre la.
  3. **Verifier les comptes Google/Apple** connectes au tel.
  4. **Hotline violences conjugales 3919** propose un soutien
     specialise pour les victimes de stalkerware.
  5. Si confirmation : **factory reset** (effacement total) du tel
     + nouveau compte Google/Apple. Eviter de restaurer depuis
     une sauvegarde (qui peut contenir le malware).

  **Cas reels :**

  - **Khashoggi 2018** : avant son assassinat, son entourage etait
    sur Pegasus. Communications interceptees.
  - **Le Monde / Mediapart 2021** : journalistes francais sur Pegasus
    selon Forbidden Stories investigation.
  - **Catalonian leaders 2022** : 65 dirigeants catalans cibles par
    Pegasus operes depuis l'Espagne.
  - **Macron 2021** : tel personnel cible par Pegasus marocain.
  - **Domestic violence cases** : milliers de cas de stalkerware
    dans le contexte de violences conjugales documentes par
    Coalition Against Stalkerware.

  **La regle d'or :**

  > Pegasus = profile a haut risque (journaliste, activiste,
  > dissident, decideur). Si tu n'es pas dans cette categorie, tres
  > peu probable d'etre cible.
  >
  > Stalkerware = peut toucher tout le monde, surtout dans les
  > contextes de violence conjugale ou de harcelement professionnel.
  > Verifier regulierement les apps installees et les Device Admin.
