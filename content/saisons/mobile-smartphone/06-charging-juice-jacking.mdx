debrief: |
  Le **juice jacking** (vol via charge USB) est un risque souvent
  ignore mais reel, particulièrement dans les bornes publiques.

  **Le mecanisme :**

  Le port USB transporte 2 choses simultanement : alimentation
  electrique ET données. Quand tu connectes ton tel a un PC, il peut
  charger ET transferer des fichiers. C'est la même chose sur une
  borne USB publique compromise.

  Si la borne est :
  - Configuree par un attaquant qui a l'a piratee
  - Le cable est modifie (cable malicieux qui simule un PC)

  Le tel peut être :
  - **Lu** : extraction de photos, contacts, documents
  - **Infecte** : installation de malware
  - **Pris en otage** : attaque "OS Modification" rare mais
    documentee

  **Les 3 catégories de risque :**

  **1. Borne USB malveillante (juice jacking pur)**

  L'attaquant installe une borne dans un lieu public ou pirate
  une borne legitime. Quand tu te branches, son système tente
  une connexion data en plus de l'alimentation.

  Cas reel : DEF CON 2011 a installé "Wall of Sheep" pour
  illustrer le juice jacking - milliers de connexions detectees,
  beaucoup acceptaient automatiquement le partage data.

  **2. Cable USB modifie (BadUSB)**

  Plus subtil : le cable LUI-MEME est piege (microcontroleur
  cache). Vendu dans des marketplaces (Hak5 O.MG Cable a 180 USD)
  ou abandonne dans des lieux stratégiques.

  Capacites :
  - Simule un clavier USB (HID injection) : tape commands sur ton
    PC quand tu le branches
  - Pas detectable visuellement (cable tout a fait normal)
  - Vecteur d'attaque ciblee (cable laisse "par hasard" pres de la
    cible)

  **3. Bornes "publiques" mal configurees**

  Pas malveillantes intentionnellement, mais mal entretenues :
  - Borne d'aeroport piratee par un attaquant a un autre vol
  - Borne de cafe avec cable use qui injecte des comportements
    aleatoires

  **Defenses (par ordre d'efficacite) :**

  **Defense 1 - Batterie externe (la meilleure)**

  Une batterie externe 10 000 - 20 000 mAh (~25 EUR) charge ton
  tel 2-3 fois sans dependre d'aucune borne. C'est l'option la
  plus simple et la plus sure.

  Marques fiables : Anker, Xiaomi, RAVPower. Éviter les marques
  no-name a 5 EUR (qualité douteuse + pas de protection
  surtension).

  **Defense 2 - Cable data-blocker / USB Condom**

  Adaptateur USB qui ne laisse passer QUE l'alimentation, pas les
  données. Brancher sur n'importe quelle borne sans risque.

  Modeles : PortaPow USB Data Blocker (10-15 EUR), Plugable USB
  Data Blocker. A garder dans son sac de voyage.

  **Defense 3 - Cable PROPRE + prise murale**

  Si la borne expose une prise murale (pas seulement des cables
  USB), c'est plus sur. Tu utilises ton propre chargeur + cable
  USB propre. Les prises murales (220V France) ne peuvent pas
  faire de transfert de données.

  **Defense 4 - Refuser le partage data sur le tel**

  Quand tu connectes ton tel, **refuse** le partage data si la
  popup apparait :
  - **iOS** : "Faire confiance a cet ordinateur ?" → **Ne pas
    faire confiance**.
  - **Android** : "Charger uniquement" → choisir cette option (vs
    "Transferer fichiers" ou "MTP").

  Si la popup n'apparait pas (Android plus ancien) : le tel est
  peut-être déjà en mode transfert par defaut. Vérifier dans les
  paramètres.

  **Defense 5 - Mode avion + écran verrouille**

  Pendant la charge, mettre le tel en mode avion + écran
  verrouille. Reduit les surfaces d'attaque (BLE, WiFi
  malveillant a portée, etc.).

  **Le piege "cable trouvé"**

  Tu trouves un cable USB Lightning / USB-C "abandonne" dans
  un cafe / bureau / aeroport. Reflexe naturel : le prendre, ça
  va depanner.

  **NE JAMAIS UTILISER UN CABLE TROUVE**. Vecteur d'attaque
  classique. Le cable peut être :
  - O.MG Cable (cable HID injection vendu sur le web)
  - BadUSB cable (firmware modifie)

  Solution : poubelle, ou si suspicieux, signaler a la sécurité
  du lieu.

  **La règle d'or pour les voyageurs :**

  > Sac de voyage cyber-safe :
  > - **Batterie externe 10 000 mAh** (25 EUR, charge 2-3 fois)
  > - **Cable data-blocker** (15 EUR, branchement sur n'importe
  >   quelle borne)
  > - **Tes propres cables** (USB-C, Lightning) en double
  > - **Chargeur secteur compact** pour prises murales
  > - **Adaptateur prise UK/US** si voyage international

  Cout total : ~80 EUR. ROI très élève quand tu evites un
  juice jacking ou un cable BadUSB.

  **Cas reels :**

  - **Los Angeles District Attorney 2019** : alerte officielle aux
    voyageurs sur le juice jacking dans les aeroports US.
  - **Chicago O'Hare 2023** : enquête federale sur des bornes
    USB piegees dans certains terminals.
  - **DEF CON Wall of Sheep** (annuel depuis 2011) : illustre
    publiquement le risque.

  Le risque reel reste statistiquement faible (estime < 0.1% des
  bornes publiques compromises selon FTC), mais les conséquences
  d'une infection (malware persistant, exfiltration de données)
  sont severes.

  **Pour les pros très mobiles :**

  Une politique BYOD/MDM peut interdire le branchement sur
  bornes USB inconnues et imposer batterie externe + cable
  data-blocker fournis par l'entreprise.

  C'est devenu une bonne pratique standard dans les secteurs
  sensibles (defense, sante, finance, journalistes
  d'investigation).
