debrief: |
  Le partage de connexion mobile est **la defense la plus simple, la
  plus efficace et la plus économique** en mobilite. Il est tellement
  supérieur aux WiFi publics qu'il devrait être le DEFAUT mobile.

  **Pourquoi c'est plus sur :**

  - Ton opérateur mobile (Orange, SFR, Bouygues, Free, MVNO) a une
    infrastructure dediee. **Pas spoofable** par un attaquant local
    (qui ne peut pas se faire passer pour Orange).
  - La connexion entre ton tel et le pylone opérateur est **chiffree
    par defaut** (LTE/5G). Personne ne peut sniffer entre les deux.
  - Pas de portail captif manipulable, pas de DNS hijack local.

  **Comment activer (procédure) :**

  **iOS (iPhone) :**
  - Reglages > Cellulaire > Partage de connexion > "Autoriser
    d'autres utilisateurs"
  - Mot de passe WiFi defini (a personnaliser, éviter le defaut auto-genere)
  - Le hotspot apparait dans la liste WiFi de ton laptop avec le nom
    de ton tel ("iPhone de Florian")

  **Android :**
  - Parametres > Réseau et Internet > Hotspot et partage de connexion
    > Hotspot WiFi
  - Personnalise le nom et le mot de passe
  - Active

  **Connexion :**
  - Sur ton laptop : choisir le hotspot dans la liste WiFi
  - Saisir le mot de passe
  - C'est tout

  **Variantes :**
  - **USB tethering** : cable USB tel-laptop. Plus rapide que WiFi, ne
    consomme pas de batterie tel.
  - **Bluetooth tethering** : plus lent, mais possible si le WiFi est
    sature.

  **Combien ça consomme ?**

  Estimations realistes :
  - **Email texte** : ~10 KB / email = 0.5 Mo pour 50 emails
  - **Email + PJ moyenne (5 Mo)** : 5 Mo / email
  - **Pages web texte** : 1-3 Mo / page
  - **YouTube 480p** : ~250 Mo / heure
  - **Visio Zoom/Teams** : ~600 Mo / heure
  - **Telechargement gros fichier** : selon taille

  Pour une session bureau distant typique (email, web, 1-2 docs) :
  **100-300 Mo / heure**.

  Forfait mobile FR moderne :
  - Forfait base 5 GB / mois : suffisant pour 15-30h de bureau distant
    par mois, mais a surveiller.
  - Forfait 100 GB / mois (Free, RED, Sosh) : largement suffisant.
  - Forfait illimite : aucun problème (sauf fair-use a 200-300 GB chez
    Free).

  **Optimisations data :**

  - **Éviter visio/streaming** sur 4G partage si tu peux (consomme
    vite).
  - **Activer mode économie data** sur Slack, Teams, Outlook (pas de
    download auto des PJ).
  - **Travailler offline** : retirer les PJ sur les emails que tu vas
    traiter (Outlook a un mode "télécharger headers seulement").

  **Cas particulier : roaming a l'étranger**

  - **UE/EEE** : "roam like at home" - même tarif qu'en France. Pas
    de problème, partage de connexion comme en France.
  - **Hors UE** : peut être cher (5 a 10 EUR / Mo). Vérifier le forfait
    avant de partager. Souvent un pass voyage est plus économique.

  **Le piege : confondre partage et fonctionnalite**

  "Partage de connexion" = 4G/5G de ton tel partage avec d'autres
  appareils. C'est sur.

  "Bouton WiFi du tel" = tu cherches un WiFi alentour pour t'y
  connecter. Pas sur (cf. ep 01-02).

  Vérifier que tu as bien active le PARTAGE (icone de chaine ou
  cercles concentriques) et pas la SCAN WiFi.

  **Pour les PME : forfait dedie**

  Pour un employé en mobilite (commercial, dev en client),
  l'employeur peut souscrire un **forfait data dedie** sur ton tel
  pro (200 GB pour ~30 EUR/mois). Cout faible vs un incident de
  sécurité par WiFi public compromis.

> sa propre carte SIM data est une option. Forfait dedie 100 GB pour
> ~10 EUR/mois. Plus stable qu'un partage tel + ne vide pas la
> batterie de ton téléphone.
