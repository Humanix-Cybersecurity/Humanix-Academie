debrief: |
  L'usage du **mail perso au bureau** est un sujet sensible
  qui melange :

  - **Vie privée** de l'employé (Code du travail, jurisprudence)
  - **Sécurité IT** de l'entreprise
  - **Surveillance** legitime par l'employeur
  - **RGPD** (données personnelles de l'employé)

  La règle générale : **separation pro/perso** pour éviter
  tous les risques.

  **Le cadre juridique :**

  **Cour de cassation 2001 - Arrêt Nikon**

  Etablit que :
  - L'employé a droit a la vie privée, **MEME** au bureau
  - Les mails identifies "personnel" ne peuvent pas être lus
    par l'employeur (sauf circonstances exceptionnelles +
    présence de l'employé + autorisation judiciaire)
  - L'usage du mail PRO pour des affaires personnelles est
    tolere mais encadre

  **Cour de cassation 2018 - Arrêt SNCF**

  Precise :
  - Les fichiers / mails par defaut sont reputes professionnels
  - L'employeur peut les consulter sans présence de l'employé
  - SAUF s'ils sont marques "personnel"

  **Donc** :
  - **Mail perso (Gmail, Outlook perso)** : protege par la
    vie privée
  - **Mail pro avec mention "personnel"** : protege
  - **Mail pro sans mention** : reputee pro, consultable

  **Les risques d'utiliser le mail perso au bureau :**

  **Risque 1 : Surveillance réseau (NON les contenus)**

  Mêmes si l'employeur ne peut pas LIRE tes mails perso, il
  peut **VOIR** :
  - Les destinations (gmail.com, outlook.live.com)
  - Les volumes de trafic
  - Les heures d'usage
  - **Pas le contenu** (HTTPS chiffre)

  Conclusion : l'employeur sait que tu utilises Gmail mais ne
  voit pas le contenu.

  **Risque 2 : Phishing perso = compromission pro**

  Tu cliques sur un phishing dans Gmail au bureau :
  - Le malware se téléchargé sur le poste PRO
  - Acces au réseau pro de l'entreprise
  - Compromission des données pro

  **Frequence** : 5-10% des breaches PME viennent de ce vecteur.

  **Risque 3 : Confusion / fuite**

  - Tu copies-colles par erreur des données pro dans un mail
    perso
  - Tu partages des fichiers pro via ton Drive perso
  - Données pro chez un tiers (Google) sans contrat

  **Risque 4 : Distraction**

  - Notifications constantes
  - Productivite degradee
  - Interruptions tout au long de la journée

  **Risque 5 : Image / climat**

  - Ton manager voit que tu passes 20% du temps sur Gmail
  - Suspicion (justifiee ou non) sur ton implication

  **Les bonnes pratiques :**

  **Pratique 1 : Téléphone perso (4G)**

  - Tes affaires perso sur **ton téléphone perso** avec **ta**
    connexion 4G (pas le wifi pro)
  - Aucun monitoring possible par l'entreprise
  - Vie privée preservee

  **Cout** : forfait 4G personnel (déjà existant).

  **Pratique 2 : Pause dediee perso**

  - Pendant pauses cafe / dejeuner
  - Hors heures de travail (avant/après)
  - **Pas pendant** la concentration pro

  **Pratique 3 : Boîte "perso vital"**

  - Pour les vraies urgences (médecin, famille, banque)
  - Acces rapide depuis le smartphone perso
  - Ne pas tout consulter, juste les urgents

  **Pratique 4 : Separation des comptes**

  - **Compte Google pro** + **compte Google perso** :
    différents
  - **Pas de profile Chrome** partage entre pro et perso
  - **Pas de sync auto** des bookmarks/passwords

  **Pratique 5 : Configuration anti-fuite**

  - Pas de copy/paste rapide entre fenêtres
  - Pas de "share to" entre apps perso et pro
  - Vérification mentale avant d'envoyer

  **Que peut faire / ne pas faire l'employeur :**

  **Peut faire** :
  - **Surveillance réseau** (volumes, destinations, pas
    contenu) avec information CSE
  - **Filtrage URL** : bloquer Gmail / Facebook au bureau
    (après consultation CSE)
  - **Logs d'acces** au SI pro
  - **Charte informatique** definissant les usages
  - **Consultation des mails pro** (sauf mention "personnel")

  **Ne peut pas faire** :
  - **Lire les mails perso** (Gmail, Outlook perso)
  - **Decrypter le HTTPS** sans accord judiciaire
  - **Surveillance massive** sans information CSE / CNIL
  - **Acces aux comptes perso** (mot de passe vole)
  - **Sanctionner** l'usage occasionnel modere du mail perso

  **Exception : circonstances exceptionnelles**

  En cas d'incident grave (suspicion de fuite, vol de données) :
  - Acces aux mails perso possible avec autorisation judiciaire
  - Présence de l'employé + huissier + IT
  - Procédure très encadree (Code travail + Code procédure
    civile)

  **Le piege "je peux tout faire chez moi" :**

  Beaucoup d'employés pensent : "comme c'est ma vie privée,
  je peux tout utiliser". MAIS :

  - **Pas de monitoring** = pas de risque sanction directe
  - **Mais risques techniques** (phishing, confusion, fuite)
  - **Et image** : même sans monitoring, le manager voit le
    temps passe

  Solution : usage modere, vraiment urgent.

  **Le piege "le DSI lit mes mails perso" :**

  Faux dans 99% des cas :
  - **Techniquement** complique (HTTPS chiffre)
  - **Juridiquement** interdit (sauf circonstance exceptionnelle)
  - **Pratiquement** : le DSI a autre chose a faire

  Ce que le DSI VOIT :
  - Tu vas sur gmail.com (volume + durée)
  - Tu téléchargés des PJ
  - Tu envoies / recois (volume seulement)

  Pas le contenu.

  **Le piege "je n'ai rien a cacher" :**

  Argument très fréquent mais errone :
  - **Ce n'est pas a l'entreprise** de decider ce qui doit
    être cache ou pas
  - **Vie privée** = droit fondamental
  - **Risques techniques** quand même (phishing, confusion)

  Solution : separer non pas par pudeur mais par hygiene.

  **Configuration recommandee :**

  **Sur ton téléphone perso** :
  - Mail perso (Gmail, Outlook perso)
  - WhatsApp / Signal perso
  - Banque perso
  - Sante perso
  - Connecte via **ta** 4G (pas le wifi pro)

  **Sur ton ordinateur pro** :
  - Mail pro
  - Outils pro (CRM, ERP, Office)
  - **Eventuellement** : mail perso pour vraies urgences,
    via navigateur prive

  **Sur ton ordinateur perso** :
  - Tout perso
  - Pas de données pro (Shadow IT, cf saison stockage-cloud
    ep 02)

  **La charte informatique :**

  Verifie ce que dit ta charte :

  - **Usage du mail perso** : autorise / restreint / interdit
  - **Niveau de monitoring** : information requise CSE
  - **Sanctions** en cas de manquement
  - **Procédures** d'incident

  **Cas reel : PME 2023 (compromission via Gmail perso)**

  Comptable utilisait Gmail perso au bureau (pratique courante).
  A clique sur un phishing dans son Gmail. Le malware s'est
  installe sur le poste pro et a acces au réseau pro :
  - Fuite de 5000 fichiers clients
  - Notification CNIL + amende
  - Cout total : 80 K EUR

  Cause : usage de Gmail perso non securise au bureau.
  Solution post-incident : Gmail bloque sur poste pro,
  encouragement usage téléphone perso pour perso.

  **Cas reel : Cabinet 2022 (RGPD)**

  Avocat envoyait des documents clients sensibles via son Gmail
  perso (plus pratique). Compte Gmail compromis (mot de passe
  reutilise). 200 dossiers clients exposes.

  Sanctions :
  - CNIL : 30 K EUR
  - Ordre des avocats : sanction professionnelle
  - Reputation : presse local

  **Cas reel : Manager 2024 (sanction injustifiee annulee)**

  Manager licencie pour "usage excessif du mail perso au
  bureau". Recours aux prud'hommes : tribunal donne raison au
  manager (usage modere, pas d'impact prouve sur son
  travail).

  Sanction annulee + dedommagement.

  Lecon : la sanction n'est pas automatique. Doit être
  proportionnee + prouvee.

  **Recommandation simple en 1 phrase :**

  > "Mail perso = téléphone perso + 4G perso. Mail pro =
  > ordinateur pro + réseau pro. Si vraiment besoin d'un mail
  > perso au bureau, modere et navigation privée."

  **La règle d'or :**

  > **Mail perso au bureau = autorise mais a risques techniques.
  > Solution simple : téléphone perso + 4G. Vie privée
  > respectee + sécurité IT. La charte informatique encadre
  > sans interdire totalement (illegal). Usage modere = pas
  > de problème.**
