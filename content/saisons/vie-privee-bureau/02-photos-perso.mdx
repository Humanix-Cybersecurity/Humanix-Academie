debrief: |
  Le **stockage de données personnelles** sur les outils de
  l'entreprise (OneDrive, SharePoint, Drive d'entreprise) est
  une pratique très fréquente mais problematique :

  - **Très fréquent** : 60% des employés ont au moins quelques
    fichiers perso sur leur Drive pro (etude Gartner)
  - **Très problematique** : violation principe separation
    perso/pro

  **Pourquoi c'est problematique :**

  **Problème 1 : Espace de l'entreprise**

  - OneDrive Business = paye par l'entreprise (5-25 EUR/user/
    mois)
  - Tes 5 Go de photos perso = ressources facturees a
    l'entreprise
  - Si tout le monde fait pareil : coûts importants

  **Problème 2 : Tu n'es pas propriétaire**

  - Si tu pars : OneDrive Business est ferme par l'entreprise
  - Tes photos perso peuvent être **perdues** si pas
    sauvegardees ailleurs
  - Conflit en cas de depart conflictuel

  **Problème 3 : RGPD**

  - L'entreprise a un **registre de traitement** (article 30
    RGPD) qui ne couvre PAS tes données perso
  - Stocker des données perso d'employés (par les employés
    eux-mêmes) sur outil pro = traitement non-encadre
  - **Risque** : non-conformité

  **Problème 4 : Audit / decouverte**

  - L'entreprise peut auditer ses Drives (avec information)
  - Tes photos perso peuvent être vues par DSI / RSSI
  - Photos sensibles (famille, sante) exposees

  **Problème 5 : Confusion**

  - Tu copies-colles par erreur des données pro dans tes
    fichiers perso
  - Tu partages des fichiers pro pensant que c'est perso
  - Données melangees = risque fuite

  **Problème 6 : Sécurité**

  - Si ton compte pro est compromis = tes données perso aussi
  - Si l'entreprise subit un incident = tes données perso
    affectees
  - Exposition non-souhaitee

  **Que dit la jurisprudence :**

  **Cour de cassation 2018 (Arret SNCF)** :

  - Les fichiers / mails par defaut sur outils pro sont
    reputes professionnels
  - L'employeur peut les consulter sans présence de l'employé
  - SAUF s'ils sont marques "personnel"

  **Donc** :
  - **Photos non marquees "perso"** : reputees pro,
    consultables par employeur
  - **Photos marquees "perso"** : protegees (sauf
    circonstances exceptionnelles)

  Mais le marquage ne resout pas les **problème 1, 2, 3, 4**
  ci-dessus.

  **Les bonnes pratiques :**

  **Pratique 1 : Solutions perso adaptees**

  Pour les photos perso :
  - **iCloud** (5 Go gratuit, 50 Go a 0.99 EUR/mois)
  - **Google Photos** (15 Go gratuit, 100 Go a 1.99 EUR/mois)
  - **Amazon Photos** (illimite avec Prime)
  - **Dropbox perso** (2 Go gratuit, 2 To a 9.99 EUR/mois)

  Pour les documents perso :
  - **iCloud Drive / Google Drive perso**
  - **Dropbox perso**
  - **NAS personnel** (Synology a la maison)

  **Pratique 2 : Separation stricte**

  - **Compte perso** = données perso
  - **Compte pro** = données pro
  - **JAMAIS** de melange

  **Pratique 3 : Charte d'entreprise**

  La charte informatique definit :
  - Tolerance de quelques fichiers perso temporaires
  - Quantite max acceptable
  - Procédure si depart (recuperation des fichiers perso)

  Souvent : **tolerance** mais **pas droit acquis**.

  **Pratique 4 : Migration en cas de bordel**

  Si tu as accumule beaucoup de perso sur le pro :
  1. Audit : qu'est-ce qui est perso ?
  2. Migration : vers solution perso adaptee
  3. Suppression : du compte pro
  4. **Vérification** après migration

  **Pratique 5 : Politique departement IT**

  L'entreprise peut :
  - **Tolerer** un peu (pragma)
  - **Encadrer** (max 1 Go, suppression après depart)
  - **Interdire** (politique stricte)

  **Selon contexte / culture / secteur.**

  **Le piege "j'ai paye mon abonnement Microsoft 365" :**

  Faux. C'est **l'entreprise** qui paye ton OneDrive Business
  (compris dans la licence employé). Tu as un usage
  professionnel, pas un droit de stockage personnel.

  **Le piege "je vais le marquer 'perso' et c'est OK" :**

  Le marquage protege la confidentialité (l'employeur ne peut
  pas LIRE), mais pas :
  - Le **stockage** (l'entreprise paye)
  - Le **comportement de depart** (suppression possible)
  - La **conformité RGPD** (usage detourne)

  **Le piege "tout le monde le fait" :**

  Pratique courante = pas pratique acceptable. La conformité
  RGPD + politique entreprise s'applique a tous.

  **Le piege "ce sont mes photos famille, c'est ma vie privée" :**

  Tu as DROIT a ta vie privée (Code travail). Mais tu n'as
  PAS droit d'utiliser les outils pro pour la stocker. Vie
  privée != stockage perso sur outils pro.

  **Cas fréquent : depart d'employé**

  Quand un employé quitte :
  1. Compte OneDrive ferme après 30-90 jours
  2. Données pro recuperees (cf saison cyber-rh ep 02)
  3. **Données perso** : que faire ?
     - Restituer ? (procédure complexe)
     - Supprimer ? (perte pour l'employé)
     - Donner option a l'employé avant fermeture ?

  Sans politique claire = conflits fréquents.

  **Politique recommandee :**

  Charte informatique d'entreprise type :

  > **Article X - Données personnelles sur outils pro** :
  > 
  > Les outils mis a disposition (OneDrive, SharePoint,
  > Outlook) sont destines a un usage professionnel. Un usage
  > personnel **occasionnel et modere** est tolere
  > (notifications, agenda mixte). Le stockage massif de
  > données personnelles (photos, archives) est **deconseille**
  > pour les raisons suivantes :
  > - Espace de stockage facturé a l'entreprise
  > - Perte potentielle des données lors du depart
  > - Confusion avec données professionnelles
  > 
  > En cas de depart, l'employé a 30 jours pour migrer ses
  > données personnelles vers une solution personnelle. Au-dela,
  > le compte sera ferme et les données supprimees.

  Equilibre : tolerance + responsabilisation + procédure de
  depart.

  **Cas reels :**

  **Cas 1 - PME 2023 (audit espace stockage)**

  Audit DSI : 30% du stockage Microsoft 365 est constitue de
  données personnelles (photos, archives, fichiers privates).
  Cout : 5K EUR/mois sur 50K total.

  Action :
  - Communication interne (3 mois)
  - Migration accompagnee (tutoriels iCloud/Google Photos)
  - Suppression après délai

  Résultat : 80% du stockage perso retire. Cout reduit
  de 4K EUR/mois.

  **Cas 2 - Depart conflictuel 2024 (procès)**

  Employe licencie en colere. Compte ferme après 30 jours.
  Reclame ses photos famille (10 Go) qu'il avait sur OneDrive
  pro. Pas de procédure de migration prévue.

  Conflit aux prud'hommes :
  - Tribunal demande restitution des données perso
  - Entreprise doit reactiver compte temporairement
  - **Cout** : 5K EUR de procédures + temps DSI

  Lecon : politique claire de fin d'emploi avec migration
  obligatoire.

  **Cas 3 - PME 2022 (RGPD)**

  Audit RGPD interne : decouverte de 50 photos d'employés en
  vacances stockees sur SharePoint d'équipe (mises par les
  employés). Données perso sans base legale.

  Action :
  - Suppression imediate
  - Information CNIL volontaire
  - Charte informatique mise a jour
  - Formation RGPD pour tous

  Pas de sanction (action proactive + bonne foi).

  **Configuration recommandee :**

  **Pour les employés** :
  - **iCloud / Google Photos perso** pour photos famille
  - **Dropbox / Google Drive perso** pour documents perso
  - **NAS perso** pour gros volumes
  - **Pas de stockage** dans OneDrive Business / SharePoint

  **Pour l'entreprise** :
  - **Politique** claire dans la charte
  - **Quota** raisonnable (par exemple 1 Go pour données mixed)
  - **Procédure de depart** : 30 jours pour migrer le perso
  - **Tutoriels** : aide aux employés pour migrer
  - **Audit** annuel des espaces

  **Le mail Outlook : même principe**

  - **Mails pro** dans la boîte pro
  - **Mails perso archives** : a transferer vers compte perso
  - **Limite** : usage occasionnel modere du mail pro pour
    des urgences perso (médecin, banque) tolere

  **Outils de migration faciles :**

  **De OneDrive a iCloud** :
  1. Telecharger photos vers PC (télécharger zip)
  2. Importer dans iCloud Photos via Mac/iPhone
  3. Supprimer du OneDrive

  **De OneDrive a Google Photos** :
  1. Telecharger photos vers PC
  2. Upload sur photos.google.com
  3. Supprimer du OneDrive

  **Outils** : Microsoft Mover (gratuit), CloudHQ, MultCloud

  **Cout perso minimal** :
  - **iCloud 50 Go** : 0.99 EUR/mois
  - **Google One 100 Go** : 1.99 EUR/mois
  - **Amazon Prime** : 5.99 EUR/mois (avec photos illimitees)

  **La règle d'or :**

  > **Données perso = solution perso (iCloud, Google Photos,
  > Dropbox perso). Données pro = outils pro (OneDrive
  > Business, SharePoint). Separation stricte = simple +
  > legal + securise. Tolerance entreprise pour quelques
  > fichiers temporaires, pas droit a stockage massif.**
